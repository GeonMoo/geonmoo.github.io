值得推荐 | Awesome IOS Software

发表于 2018-02-25 更新于 2020-02-02 分类于 Treasure 私家珍藏 Valine：

最后更新时间：2018年2月25日

由于笔者目前只使用iPad，因此此处只进行iPad软件的推荐。敬请期待IOS的其它硬件平台软件的更新。欢迎讨论交流！Email: mozheyang@outlook.com

iPad除了可以是一个娱乐工具，同时也可以成为我们学生党、磕盐汪忠实而有效的学习工具、效率工具。今天笔者就给大家推荐两款付费App…买了一直用，真真是真爱！

MarginNote

在浮躁的社会，我们拿起电子产品就想要刷社交软件和游戏。就算是想要认真读上一篇文章，却也总会觉得电子产品显示区域的不足的缺点，难以给我们的思想提供任意翱翔的场所。

思维导图是深入理解知识结构的利器。笔者第一次了解到思维导图这种思维工具是在大学读了东尼.伯赞（Buzan T.）的思维导图系列丛书之后了。现已经有了不少思维导图构造软件如Xmind、MindManager，网页App中也有百度脑图等可以推荐。

但上述构造思维导图的方式都滞后于阅读。也就是说，我们需要先阅读，甚至有了一定的理解之后，才能将看似混乱的逻辑知识点整理成为思维导图，最后使用上述软件输出。笔者这次给大家推荐的软件，正是针对这个问题，提出了——阅读过程“收集”，阅读过后“整理”的解决方案。整个书籍/文献的阅读学习过程变得一气呵成，从而大大提高我们阅读学习的效率。

目前该软件只有iPad和Mac版本，也就是说Windows PC用户暂时还无法享受到如此好用的一款软件。据说该软件作者（中国人！）已经在着手开发Windows版本的MarginNote，或许2020年的大学生就有机会用上了啊。

笔者认为现今Windows PC也只有Docear这个开源软件可能与之匹敌，详情请关注笔者博文：思维导图+文献管理 | Docear使用体会。

Wordflex

接触过大量单词背诵（四/六级、托福/雅思、GRE，尤其是GRE）的同学可能会感受到这样一些困难：单词近义词太多，导致记忆混乱；单词意义太过抽象，难以理解；单词书太丑，缺乏艺术感 :)

单词释义记忆混乱主要是因为近义词与反义词记忆的混乱。在使用中文理解英文单词时，可能会发现好像好几个单词的中文释义相近甚至相同，但是在英文语境中却不能互换。这时候一个可以表现近义词与反义词联系的工具就会显得很重要。

在寻找类似单词网络工具的过程中，笔者还发现了WordNet项目与一款名叫Visual Thesaurus的网页App。但最后选择了Wordflex。大家可以很明显地看到它在界面上的优势，使用之后还会感叹他在词库（牛津词典）上的优势。

通过近义词和反义词，我们可以更立体地理解一个英文单词。在Wordflex中，我们可以通过一个单词从而了解一批近义词——通过轻松的点击（Tap）就可以进入下一个单词的单词网络，超值！

目前该软件只有iPad版本，其他平台可以考虑网页软件Visual Thesaurus. 该网站免费版似乎有查询次数限制，总之还是付费软件更舒心啦。买苹果就为了它平台上的几个好用的软件，也算是值了吧。
