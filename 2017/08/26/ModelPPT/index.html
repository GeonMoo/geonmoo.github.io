任何事情都没那么简单——尤其是你真的想把它做好的时候

发表于 2017-08-26 更新于 2026-03-10 本文字数： 541 阅读时长 ≈ 1 分钟

快开学了。发一篇半心情半展示的日志吧。

大四的时候，很幸运地，笔者能够加入到一个优秀的团队，做一件很酷的事情——给低年级本科生同学“讲课”，介绍数学建模经验以及相关背景知识。

那时的我也是第一次，体验了如教师一般，从自己了解掌握知识，再尝试将其与其他同学分享的那种既痛苦又享受的过程。

其实这并不矛盾。

“痛苦”是因为我把这个任务当做我自己的艺术作品来对待，我需要耗费大量的精力，来打磨其细节。一份PPT只进行两节2h时长的授课，前前后后却将近花费了我们2个月时间仔细揣摩和修改。笔者不敢说最后大家听的热情有多高，讲说的效果有多好，但至少我自己认为：它的确成为了一份值得收藏的工艺品。

而“享受”是因为我在整个过程中，始终把这不多的四堂课当做自己的作品，可以跟随自己一辈子的作品。我可以去想象作品完成时的成就感，也可以去想象他们可能给有需要的人们所带来的帮助。在不为物质生活担忧的前提下，这种精神层面上的社会、个人价值实现和自我认同所带来的满足感，实在是一剂“兴奋剂”。

笔者对自己的职业规划里有“知识分享者”这一选择。这意味着，我可以享受成为一个可能的“传道授业解惑”的教师职业。至于未来到底是什么样子，交给未来去想吧。

最后，非常欢迎各位有想法的朋友们与我联系。
只要你主动，我们就会有故事 :)
